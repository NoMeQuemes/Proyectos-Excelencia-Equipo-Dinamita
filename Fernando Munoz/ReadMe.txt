Descripción del proyecto:

Se realizó un CRUD simulando una Todo List (Lista de Tareas) en donde el usuario puede crear tareas nuevas, editarlas y eliminarlas. 
El back-end se realizó con C# en conjunto con .NET, y se utilizó el ORM Entity Framework para crear y hacer el mapeo de la BD.
El front-end fue desarrollado con el framework VUE en su versión 3. Además, se utlizó la librería Axios para consumir la API

Hostings:
